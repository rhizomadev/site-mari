# Site Mariana Strelow — Psicoterapia

Site estático de página única para a psicóloga Mariana Strelow.

## Como rodar

```bash
npm install
npm run dev
```

Acesse [http://localhost:4321](http://localhost:4321).

## Build para produção

```bash
npm run build
```

Os arquivos estarão em `dist/`.

## Tecnologias

- Astro 4
- CSS com variáveis e animações
- Design responsivo (mobile-first)
